# RINDA AI — B2B 수출 영업 자동화 플러그인

> **RINDA AI**는 해외 바이어를 찾고, 연락처를 확보하고, 이메일 캠페인을 만들고, 답변을 관리하는 과정을 **AI가 대신 처리**해주는 Claude Code 플러그인입니다.

수출 영업에 필요한 반복적인 작업을 자동화하여, **영업 담당자가 실제 거래 성사에만 집중**할 수 있도록 도와줍니다.

---

## 이런 분들에게 추천합니다

- 해외 바이어를 직접 찾아야 하는 **수출 영업 담당자**
- 다수의 해외 거래처에 이메일 아웃리치를 해야 하는 **무역 회사**
- B2B 영업 프로세스를 자동화하고 싶은 **스타트업 및 중소기업**
- 수출 캠페인의 성과를 한눈에 파악하고 싶은 **영업 관리자**

---

## 주요 기능

### 1. 해외 바이어 검색 (`/rinda-ai:buyer-search`)

**원하는 산업과 국가를 입력하면, AI가 적합한 해외 바이어를 자동으로 찾아줍니다.**

- 산업군(예: 화장품, 기계류)과 대상 국가(예: 미국, 독일)를 지정
- 수입업체(Importer), 유통업체(Distributor), 도매업체(Wholesaler) 등 바이어 유형 선택 가능
- 최소 매출 규모, 검색 수량 등 필터링 조건 설정
- AI가 발견한 바이어를 **점수 기반으로 자동 랭킹**하여 우선순위를 제시
- 검색 완료 후 바로 연락처 확보 또는 이메일 캠페인 생성으로 이어갈 수 있음

> **예시**: "미국에서 화장품을 수입하는 연매출 500만 달러 이상의 바이어 50개를 찾아줘"

---

### 2. 연락처 자동 확보 (`/rinda-ai:enrich`)

**찾은 바이어의 담당자 이메일, 전화번호, LinkedIn 프로필 등을 자동으로 수집합니다.**

- 바이어 검색 결과에서 한 번에 최대 100개 리드의 연락처를 확보
- 이메일, 전화번호, LinkedIn 프로필, 취급 제품 정보 등 수집
- **의사결정자(구매 담당자, 이사 등)를 자동으로 식별**하여 우선순위 분류
  - **높은 우선순위**: 구매 담당자 이메일 확보 + 제품 매칭
  - **보통 우선순위**: 이메일 확보했으나 직책 불명확
  - **낮은 우선순위**: 대표 이메일(info@, contact@)만 확보
- 배치(묶음) 처리로 안정적이고 빠른 데이터 수집

> **예시**: "방금 찾은 바이어들의 연락처를 확보해줘"

---

### 3. AI 이메일 캠페인 생성 (`/rinda-ai:sequence-create`)

**AI가 여러 단계의 영업 이메일을 자동으로 작성하고, 바이어에게 순차적으로 발송합니다.**

- 캠페인 이름, 톤(전문적/친근/격식체 등), 언어, 이메일 단계 수 설정
- AI가 각 단계의 **제목과 본문을 자동 생성** (개인화 토큰 포함: 이름, 회사명 등)
- 생성된 이메일을 **미리보기로 확인**하고, 승인/수정/재생성 선택 가능
- 승인 후 선택한 바이어에게 **자동 등록 및 스케줄링**
- 이메일 작성 가이드라인 기반의 품질 보장:
  - 제목: 60자 이내, 회사명 포함
  - 본문: Hook(니즈 언급) → Value(제안) → CTA(구체적 행동 유도)
  - 팔로업: 매번 새로운 가치 제공, 점진적 톤 변화

> **예시**: "US Cosmetics Q1 Outreach라는 이름으로 6단계 영업 이메일 시퀀스를 만들어줘"

---

### 4. 답변 관리 및 우선순위 분류 (`/rinda-ai:reply-check`)

**바이어가 보낸 답변을 자동으로 분석하여 어떤 조치를 취해야 하는지 알려줍니다.**

- 읽지 않은 답변을 자동으로 가져와 의도(intent)별로 분류:
  - **미팅 요청** → 즉시 응답 (미팅 시간 제안)
  - **긍정적 관심** → 당일 응답 (카탈로그 발송, 통화 제안)
  - **질문** → 질문 답변 + 3일 후 팔로업
  - **지금은 아님** → 90일 후 재접근
  - **거절** → 태그 업데이트 후 시퀀스에서 제외
  - **부재중** → 복귀 후 팔로업 재개
  - **수신 거부** → 즉시 수신 거부 처리
- **긴급도별 정렬**로 가장 중요한 답변부터 처리 가능

> **예시**: "오늘 온 답변들을 확인하고 무엇을 해야 하는지 알려줘"

---

### 5. 캠페인 성과 리포트 (`/rinda-ai:campaign-report`)

**캠페인의 전체 성과를 한눈에 파악할 수 있는 보고서를 생성합니다.**

- **퍼널 분석**: 리드 발견 → 연락처 확보 → 이메일 발송 → 답변 → 미팅 예약
- **이메일 성과**: 발송 수, 오픈율, 클릭율, 답변율, 바운스율
- **핫 리드 목록**: 즉시 대응이 필요한 유망 바이어 목록
- **업계 벤치마크 비교**:
  - 오픈율: 35% 이상 양호, 50% 이상 우수
  - 답변율: 10% 이상 양호, 20% 이상 우수
  - 바운스율: 5% 이하 건강함
- **AI 인사이트**: 성과 기반 개선 제안 자동 생성
- 특정 시퀀스 단위로 드릴다운 분석 가능 (7일/30일/90일 기간 선택)

> **예시**: "지난 30일간의 캠페인 성과를 보여줘"

---

### 6. 계정 연결 (`/rinda-ai:connect`)

**Google 계정으로 간편하게 RINDA AI에 로그인합니다.**

- 최초 1회만 실행하면 됨
- Google OAuth를 통한 안전한 인증
- 토큰 자동 갱신으로 매번 로그인할 필요 없음 (14일간 유효)
- 연결 상태 확인 및 로그아웃 기능 제공

---

## 자동 워크플로우 연결

각 기능은 독립적으로 사용할 수 있지만, **자연스럽게 다음 단계로 이어지도록 설계**되어 있습니다:

```
바이어 검색 → 연락처 확보 → 이메일 캠페인 생성 → 답변 관리 → 성과 리포트
```

각 단계가 끝나면 AI가 다음 단계를 제안하고, 동의하면 이전 결과(바이어 ID, 시퀀스 ID 등)를 자동으로 넘겨받아 진행합니다.

---

## 시작하기

### 1단계: 플러그인 설치

```bash
# Claude Code에서 플러그인 설치
/plugin marketplace add FINGU-GRINDA/claude-rinda-plugin
```

### 2단계: 계정 연결

```
/rinda-ai:connect
```

Google 계정으로 로그인하면 준비 완료!

### 3단계: 바이어 찾기 시작

```
/rinda-ai:buyer-search
```

산업과 국가를 입력하면, AI가 알아서 바이어를 찾고, 연락처를 확보하고, 이메일을 보내는 것까지 안내해줍니다.

## 자주 묻는 질문 (FAQ)

### 일반

**Q: RINDA AI를 사용하려면 프로그래밍 지식이 필요한가요?**
> 아닙니다. 자연어(한국어/영어)로 대화하듯이 명령하면 됩니다. 예를 들어 "미국에서 화장품 수입업체를 찾아줘"라고 하면 AI가 알아서 처리합니다.

**Q: RINDA AI는 무료인가요?**
> RINDA AI는 구독 플랜 기반으로 운영됩니다. 플랜별로 검색 가능한 리드 수에 제한이 있으며, 캠페인 리포트에서 현재 사용량을 확인할 수 있습니다.

**Q: 어떤 산업/제품에 사용할 수 있나요?**
> B2B 수출이 가능한 모든 산업에 사용할 수 있습니다. 화장품, 식품, 기계류, 전자제품, 화학제품, 섬유 등 산업군을 자유롭게 지정할 수 있습니다.

**Q: 어떤 국가의 바이어를 찾을 수 있나요?**
> 전 세계 대부분의 국가를 지원합니다. 미국, 유럽, 동남아시아, 중동 등 원하는 타겟 시장을 지정할 수 있습니다.

---

### 계정 및 인증

**Q: Google 계정이 없으면 사용할 수 없나요?**
> 현재는 Google OAuth만 지원합니다. Google 계정으로 로그인해야 합니다.

**Q: 로그인이 만료되면 어떻게 하나요?**
> 일반적인 사용에서는 토큰이 자동으로 갱신됩니다. 14일 이상 사용하지 않은 경우에만 `/rinda-ai:connect`를 다시 실행하면 됩니다.

**Q: 내 인증 정보는 안전한가요?**
> 네. 인증 토큰은 로컬 컴퓨터에만 저장되며(파일 권한 600), 모든 통신은 HTTPS로 암호화됩니다. 토큰은 1시간마다 자동 갱신되고, 로그아웃 시 즉시 삭제됩니다.

---

### 바이어 검색

**Q: 검색 결과가 너무 적게 나오면 어떻게 하나요?**
> 검색 조건을 넓혀보세요. 최소 매출 기준을 낮추거나, 바이어 유형을 추가하거나, 인접 산업군으로 확장해볼 수 있습니다.

**Q: 이전에 연락한 바이어가 다시 검색되나요?**
> 아닙니다. 이미 연락한 바이어는 자동으로 제외됩니다.

**Q: 한 번에 몇 개의 바이어를 검색할 수 있나요?**
> 기본값은 50개이며, 필요에 따라 조정할 수 있습니다.

---

### 연락처 확보

**Q: 확보한 연락처의 정확도는 어느 정도인가요?**
> AI가 여러 소스에서 정보를 수집하고 검증합니다. 높은 우선순위로 분류된 리드는 의사결정자의 직접 이메일이 확인된 경우입니다.

**Q: 한 번에 최대 몇 개의 연락처를 확보할 수 있나요?**
> 1회 실행당 최대 100개 리드의 연락처를 확보할 수 있습니다.

---

### 이메일 캠페인

**Q: AI가 작성한 이메일을 수정할 수 있나요?**
> 네. AI가 이메일을 생성한 후 미리보기를 보여주며, 각 단계별로 승인/수정/전체 재생성을 선택할 수 있습니다.

**Q: 한국어로 이메일을 보낼 수 있나요?**
> 네. 이메일 생성 시 언어를 지정할 수 있습니다. 한국어(`ko`), 영어(`en`), 독일어(`de`), 프랑스어(`fr`) 등을 지원합니다.

**Q: 이메일이 스팸으로 분류되지 않나요?**
> RINDA AI의 이메일 작성 가이드라인은 스팸 트리거 단어(예: "무료", "지금 바로", "한정")를 자동으로 배제합니다. 또한 개인화된 내용과 적절한 발송 간격으로 스팸 가능성을 최소화합니다.

**Q: 팔로업 이메일의 발송 간격은 어떻게 되나요?**
> 기본 간격은 +3일, +7일, +14일, +30일입니다. 캠페인 생성 시 조정할 수 있습니다.

---

### 답변 관리

**Q: 답변 분류는 얼마나 정확한가요?**
> AI가 답변의 의도(미팅 요청, 관심 표현, 질문, 거절 등)를 자동으로 분석합니다. 분류 결과와 함께 구체적인 다음 행동을 제안하므로, 최종 판단은 사용자가 하게 됩니다.

**Q: "지금은 아님"이라고 답한 바이어는 어떻게 되나요?**
> 90일 후 새로운 제안이나 업데이트된 오퍼로 재접근하도록 자동 태그됩니다.

---

### 리포트

**Q: 업계 평균과 비교할 수 있나요?**
> 네. 캠페인 리포트는 B2B 수출 영업의 업계 벤치마크(오픈율 35%, 답변율 10% 등)와 자동으로 비교하여 현재 성과가 양호한지, 개선이 필요한지 알려줍니다.

**Q: 특정 캠페인만 따로 볼 수 있나요?**
> 네. 시퀀스 ID를 지정하면 해당 캠페인의 단계별 성과를 상세하게 확인할 수 있습니다.
