## 1. RINDA AI란 무엇인가?

**RINDA AI**는 해외 B2B 수출 영업을 자동화해주는 Claude Code 플러그인입니다.

기존에는 아래 작업을 사람이 직접 해야 했습니다:

| 기존 방식 | RINDA AI 방식 |
|-----------|---------------|
| 직접 검색해서 해외 바이어 찾기 | AI가 자동으로 바이어 리스트 제공 |
| LinkedIn·웹사이트 뒤져서 연락처 수집 | AI가 이메일·전화·SNS 자동 수집 |
| 영업 이메일 직접 쓰기 | AI가 단계별 이메일 시퀀스 자동 생성 |
| 답장 일일이 확인하고 우선순위 정하기 | AI가 답장 분류 & 다음 행동 제안 |
| 엑셀로 성과 집계 | AI가 리포트 자동 생성 |

**한마디로:** 챗봇에게 말하듯 지시하면, AI가 RINDA 플랫폼과 연동하여 수출 영업 전 과정을 자동으로 처리합니다.

## 2. Claude Cowork에서 어떻게 연동하는가?

### 연동 구조 이해하기

```
사용자 (대화)
    ↓
Claude Cowork (AI 대화창)
    ↓
RINDA AI 플러그인 (자동 설치됨)
    ↓
RINDA 서버 (바이어 DB, 이메일 전송, 통계)
```

Claude Cowork에서 RINDA AI 플러그인을 설치하면:

- 처음 세션 시작 시 `rinda-cli`와 `rinda-mcp` 도구가 **자동으로 설치**됩니다.
- 이후에는 그냥 채팅창에 한국어로 지시하면 AI가 알아서 처리합니다.
- 별도 프로그램 설치나 코드 작성이 전혀 필요 없습니다.

### 플러그인이 제공하는 도구 목록

내부적으로 AI가 아래 18가지 도구를 자동으로 활용합니다 (사용자가 직접 입력할 필요 없음):

| 도구 이름 | 하는 일 |
|-----------|---------|
| `rinda_auth_status` | 로그인 상태 확인 |
| `rinda_buyer_search` | 바이어 검색 시작 |
| `rinda_buyer_status` | 검색 진행 상황 확인 |
| `rinda_buyer_results` | 검색 결과 가져오기 |
| `rinda_buyer_select` | 선택한 바이어 저장 |
| `rinda_buyer_enrich` | 바이어 상세 연락처 수집 |
| `rinda_buyer_clarify` | 검색 조건 추가 답변 |
| `rinda_buyer_sessions` | 과거 검색 기록 조회 |
| `rinda_sequence_create` | 이메일 시퀀스 생성 |
| `rinda_sequence_list` | 기존 시퀀스 목록 조회 |
| `rinda_sequence_generate` | AI 이메일 내용 자동 생성 |
| `rinda_sequence_add_contact` | 바이어를 시퀀스에 등록 |
| `rinda_email_send` | 이메일 발송 |
| `rinda_reply_check` | 받은 답장 확인 |
| `rinda_campaign_stats` | 캠페인 통계 조회 |
| `rinda_workspace_list` | 워크스페이스 목록 확인 |

## 3. 처음 시작하기 — 설치 & 계정 연결

### Step 1: 플러그인 설치

Claude Cowork 채팅창에 아래를 복사해서 붙여넣으세요:

```
/plugin marketplace add FINGU-GRINDA/claude-rinda-plugin
```

> 처음 세션이 시작될 때 필요한 도구가 자동으로 설치됩니다. 설치에 약 30초가 소요됩니다.

---

### Step 2: RINDA 계정 연결 (Google 로그인)

플러그인 설치 후 아래를 입력하세요:

```
/rinda-ai:rinda 내 계정을 연결해줘
```

그러면 AI가 구글 로그인 링크를 제공합니다. 링크를 클릭하여 Google 계정으로 로그인하면 자동으로 연결됩니다.

> **보안 안내:** 로그인 토큰은 내 컴퓨터 로컬에만 저장되며 (`~/.rinda/credentials.json`), 모든 통신은 HTTPS로 암호화됩니다. 로그인은 1시간마다 자동 갱신되고, 14일간 미사용 시에만 재로그인이 필요합니다.

---

### Step 3: 로그인 상태 확인

```
/rinda-ai:rinda 내가 지금 로그인되어 있는지 확인해줘
```

## 4. 핵심 기능 5가지 & 사용법

> 아래 예시 문장을 그대로 복사해서 Claude Cowork 채팅창에 붙여넣으면 됩니다.

---

### 4-1. 해외 바이어 검색

**기능 설명:** 원하는 산업, 국가, 바이어 유형, 매출 조건을 지정하면 AI가 자동으로 해외 바이어 리스트를 찾아 순위를 매겨 제시합니다.

**평가 기준 (자동 적용):**
- 연매출 규모 (30%)
- 직원 수 규모 (15%)
- 수입 이력 (25%)
- 우리 제품과의 적합도 (30%)

#### 사용 예시

**기본 검색:**
```
미국에서 화장품 수입업체 50개를 찾아줘
```

**조건 상세 지정:**
```
독일과 프랑스에서 연매출 500만 달러 이상의 식품 수입업체를 찾아줘
```

**특정 바이어 유형 지정:**
```
일본에서 기계류를 취급하는 도매상 30곳을 찾아줘
```

**과거 검색 결과 다시 보기:**
```
지난번에 검색한 바이어 목록 다시 보여줘
```

#### 검색 후 AI가 하는 일

1. 비동기로 검색을 시작합니다 (수십 초~몇 분 소요)
2. 진행 상황을 자동으로 확인합니다
3. 검색 중 추가 정보가 필요하면 질문을 드립니다
4. 검색 완료 후 순위 테이블을 제시합니다
5. "연락처를 확보할까요?" "이메일 시퀀스를 만들까요?" 등 다음 단계를 제안합니다

---

### 4-2. 연락처 확보 (이메일·전화·LinkedIn)

**기능 설명:** 검색된 바이어들의 상세 연락처(이메일, 전화번호, LinkedIn, 취급 제품)를 자동으로 수집하고 우선순위를 분류합니다.

#### 우선순위 분류 기준

| 등급 | 조건 |
|------|------|
| **높음 (High)** | 구매 담당자 직통 이메일 + 제품 매칭 확인 |
| **보통 (Medium)** | 이메일 확보, 직책 불명확 |
| **낮음 (Low)** | 대표 이메일만 확보 (info@, contact@ 등) |

#### 사용 예시

**방금 검색한 바이어 연락처 확보:**
```
방금 찾은 바이어들의 연락처를 모두 확보해줘
```

**특정 바이어 연락처 확보:**
```
ABC Company의 연락처를 자세히 확보해줘
```

**결과 후 제안:**
```
연락처 확보가 끝나면 우선순위별로 정리하고, 이메일 캠페인을 만들어줘
```

---

### 4-3. 이메일 캠페인 만들기

**기능 설명:** AI가 여러 단계의 영업 이메일 시퀀스를 자동으로 생성합니다. 각 이메일에 바이어 회사명·제품 정보 등을 자동으로 반영합니다.

#### AI가 적용하는 이메일 작성 원칙

| 요소 | 기준 |
|------|------|
| 제목 | 60자 이내, 회사명 포함 |
| 본문 구조 | Hook(니즈 언급) → Value(제안) → CTA(행동 유도) |
| 팔로업 | 매번 새로운 가치 제공, 점점 짧게 작성 |

#### 사용 예시

**새 이메일 시퀀스 만들기:**
```
"US Cosmetics Q1"이라는 6단계 영업 이메일 시퀀스를 만들어줘
```

**기존 시퀀스 목록 보기:**
```
현재 만들어진 이메일 시퀀스 목록을 보여줘
```

**바이어를 시퀀스에 등록:**
```
방금 연락처 확보한 바이어들을 "US Cosmetics Q1" 시퀀스에 등록해줘
```

**개별 이메일 발송:**
```
buyer@company.com에게 첫 인사 이메일을 보내줘
```

---

### 4-4. 받은 답변 관리하기

**기능 설명:** 바이어들에게서 온 답장을 자동으로 확인하고, 의도와 긴급도별로 분류하여 다음 행동을 제안합니다.

#### 자동 분류 기준

| 분류 | 의미 | AI 권장 행동 |
|------|------|--------------|
| 미팅 요청 | 당장 미팅하고 싶다 | 즉시 응답 |
| 긍정적 관심 | 관심 있지만 아직 미팅 전 | 당일 응답 |
| 질문 | 제품/조건에 대해 궁금한 것이 있다 | 답변 + 3일 후 팔로업 |
| 지금은 아님 | 관심 있지만 시기가 아님 | 90일 후 재접근 |
| 거절 | 필요 없음 | 태그 업데이트, 시퀀스 제외 |
| 수신 거부 | 이메일 그만 보내 달라 | 즉시 처리 (중요) |

#### 사용 예시

**오늘 온 답장 확인:**
```
오늘 온 답장을 확인하고 무엇을 해야 하는지 알려줘
```

**최근 50개 답장 분석:**
```
최근 답장 50개를 분석해서 우선순위별로 정리해줘
```

**핫 리드 파악:**
```
지금 당장 연락해야 할 뜨거운 바이어가 누구야?
```

---

### 4-5. 캠페인 성과 리포트

**기능 설명:** 캠페인의 전체 성과를 분석하고 B2B 수출 업계 벤치마크와 비교하여 인사이트를 제공합니다.

#### B2B 수출 이메일 업계 벤치마크

| 지표 | 양호 기준 |
|------|-----------|
| 오픈율 | 35% 이상 |
| 답변율 | 10% 이상 |
| 클릭률 | 일반적으로 2~5% |

#### 조회 기간 옵션

| 입력 | 의미 |
|------|------|
| `7d` | 최근 7일 |
| `30d` | 최근 30일 |
| `90d` | 최근 90일 |
| `2w` | 최근 2주 |
| `3m` | 최근 3개월 |

#### 사용 예시

**30일 성과 보기:**
```
지난 30일간의 캠페인 성과를 보여줘
```

**주간 리포트:**
```
이번 주 이메일 캠페인 성과를 분석해줘
```

**분기 리포트:**
```
최근 3개월 성과를 업계 평균과 비교해서 알려줘
```

## 5. 처음부터 끝까지 — 전체 흐름 한눈에

아래 순서대로 진행하면 바이어 발굴부터 계약까지 전 과정을 자동화할 수 있습니다.

```
① 바이어 검색
    → "미국 화장품 수입업체 50개 찾아줘"

② 연락처 확보
    → "찾은 바이어들 연락처 확보해줘"

③ 이메일 시퀀스 생성
    → "6단계 영업 이메일 시퀀스 만들어줘"

④ 바이어 시퀀스 등록
    → "바이어들을 방금 만든 시퀀스에 등록해줘"

⑤ 답변 확인 & 대응
    → "오늘 온 답장 확인하고 다음 행동 알려줘"

⑥ 성과 분석
    → "30일 캠페인 성과 보여줘"
```

> **팁:** 각 단계를 완료하면 AI가 자동으로 다음 단계를 제안하고, 이전 단계에서 얻은 바이어 ID·시퀀스 ID 등을 자동으로 넘겨받아 진행합니다.

## 6. 자주 묻는 질문 (FAQ)

### Q. 코딩이나 기술 지식이 필요한가요?
전혀 필요 없습니다. 한국어로 대화하듯 입력하면 AI가 모든 기술적 처리를 대신합니다.

---

### Q. 어떤 나라 바이어를 찾을 수 있나요?
전 세계 대부분의 국가를 지원합니다. 미국, 독일, 일본, 중국, 동남아, 중동, 유럽 등 수출이 가능한 모든 지역에서 검색할 수 있습니다.

---

### Q. 어떤 산업을 지원하나요?
B2B 수출이 가능한 모든 산업을 지원합니다. 주요 예시:
- 화장품 / 뷰티
- 식품 / 음료
- 기계 / 제조
- 전자부품
- 의류 / 패션
- 의료기기
- 건축자재 등

---

### Q. 내 데이터는 안전한가요?
- 로그인 토큰은 내 컴퓨터에만 저장됩니다 (`~/.rinda/credentials.json`)
- 외부에서 접근할 수 없도록 파일 권한이 설정됩니다
- 모든 서버 통신은 HTTPS(암호화)로 이루어집니다
- RINDA 대시보드와 동일한 보안 수준을 사용합니다

---

### Q. 로그인이 자주 끊기나요?
아닙니다. 토큰이 자동으로 갱신됩니다. 14일 이상 완전히 사용하지 않은 경우에만 재로그인이 필요합니다.

---

### Q. 이메일이 스팸으로 분류될 수 있나요?
RINDA는 RINDA 플랫폼을 통해 이메일을 발송하며, 내장된 이메일 작성 가이드라인(제목 60자 이내, 개인화 포함 등)을 따릅니다. 단, 최종적인 발송 도달율은 RINDA 서버 설정에 따릅니다.

---

### Q. 이미 RINDA 계정이 없어도 사용할 수 있나요?
RINDA AI 기능을 사용하려면 RINDA 계정이 필요합니다. [rinda.ai](https://rinda.ai) 에서 계정을 먼저 만드세요.

---

### Q. 문제가 생기면 어디에 문의하나요?
- 홈페이지: [rinda.ai](https://rinda.ai)
- 이메일: support@grinda.ai
