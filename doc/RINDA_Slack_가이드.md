> 슬랙은 마크다운 표(table)를 지원하지 않으므로, 표 대신 목록 형식으로 작성했습니다.

## [슬랙 메시지 3] STEP 2 — 계정 연결

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔑 *STEP 2 — RINDA 계정 연결 (Google 로그인)*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

플러그인 설치 후 Claude Cowork 채팅창에 아래를 입력하세요.

```
/rinda-ai:rinda 내 계정을 연결해줘
```

그러면 AI가 구글 로그인 링크를 줍니다.
링크 클릭 → Google 로그인 → 자동 연결 완료!

🔒 *보안 안내*
• 로그인 정보는 내 컴퓨터에만 저장 (외부 접근 불가)
• 모든 통신은 HTTPS 암호화
• 토큰은 자동 갱신 (14일 미사용 시에만 재로그인 필요)
```

## [슬랙 메시지 5] 기능 1 — 해외 바이어 검색

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔍 *기능 1 — 해외 바이어 검색*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

원하는 나라, 업종, 매출 조건을 말하면 AI가 바이어 리스트를 자동으로 찾아 순위까지 매겨줍니다.

*바이어 순위 평가 기준 (AI 자동 적용)*
• 우리 제품과의 적합도 — 30%
• 연매출 규모 — 30%
• 수입 이력 — 25%
• 직원 수 규모 — 15%

*💬 이렇게 입력하면 됩니다*

기본 검색:
```
/rinda-ai:rinda 미국에서 화장품 수입업체 50개를 찾아줘
```

조건 상세 지정:
```
/rinda-ai:rinda 독일과 프랑스에서 연매출 500만 달러 이상의 식품 수입업체를 찾아줘
```

특정 바이어 유형:
```
/rinda-ai:rinda 일본에서 기계류를 취급하는 도매상 30곳을 찾아줘
```

과거 검색 다시 보기:
```
/rinda-ai:rinda 지난번에 검색한 바이어 목록 다시 보여줘
```

_검색은 수십 초~몇 분 소요. AI가 진행 상황을 자동으로 확인하고 결과를 가져옵니다._
```

## [슬랙 메시지 6] 기능 2 — 연락처 확보

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📬 *기능 2 — 연락처 확보 (이메일·전화·LinkedIn)*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

검색된 바이어의 상세 연락처를 자동으로 수집하고 우선순위를 분류해줍니다.

*우선순위 분류 기준*
• 🔴 높음 — 구매 담당자 직통 이메일 + 제품 매칭 확인
• 🟡 보통 — 이메일 확보, 직책 불명확
• 🟢 낮음 — 대표 이메일만 확보 (info@, contact@ 등)

*💬 이렇게 입력하면 됩니다*

방금 찾은 바이어 전체 연락처 확보:
```
/rinda-ai:rinda 방금 찾은 바이어들의 연락처를 모두 확보해줘
```

특정 바이어 연락처 확보:
```
/rinda-ai:rinda ABC Company의 연락처를 자세히 확보해줘
```

연락처 확보 후 바로 캠페인 연결:
```
/rinda-ai:rinda 연락처 확보가 끝나면 우선순위별로 정리하고, 이메일 캠페인을 만들어줘
```
```

## [슬랙 메시지 7] 기능 3 — 이메일 캠페인

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✉️ *기능 3 — 이메일 캠페인 만들기*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI가 여러 단계의 영업 이메일을 자동으로 작성합니다.
바이어 회사명·제품 정보 등이 이메일에 자동으로 반영됩니다.

*AI 이메일 작성 원칙 (자동 적용)*
• 제목: 60자 이내, 회사명 포함
• 본문: 니즈 언급 → 우리 제안 → 행동 유도 순서
• 팔로업: 매번 새로운 가치 제공, 점점 간결하게

*💬 이렇게 입력하면 됩니다*

새 이메일 시퀀스 만들기:
```
/rinda-ai:rinda "US Cosmetics Q1"이라는 6단계 영업 이메일 시퀀스를 만들어줘
```

기존 시퀀스 목록 보기:
```
/rinda-ai:rinda 현재 만들어진 이메일 시퀀스 목록을 보여줘
```

바이어를 시퀀스에 등록:
```
/rinda-ai:rinda 방금 연락처 확보한 바이어들을 "US Cosmetics Q1" 시퀀스에 등록해줘
```

개별 이메일 바로 발송:
```
/rinda-ai:rinda buyer@company.com에게 첫 인사 이메일을 보내줘
```
```

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📥 *기능 4 — 받은 답변 관리하기*

*자동 분류 기준 & 권장 행동*

최근 답장 분석:
```
/rinda-ai:rinda 최근 답장 50개를 분석해서 우선순위별로 정리해줘
```

뜨거운 리드 파악:
```

## [슬랙 메시지 9] 기능 5 — 성과 리포트

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 *기능 5 — 캠페인 성과 리포트*
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

캠페인 성과를 분석하고 업계 평균과 비교해서 인사이트를 제공합니다.

*B2B 수출 이메일 업계 벤치마크*
• 오픈율 → 35% 이상이면 양호
• 답변율 → 10% 이상이면 양호
• 클릭률 → 2~5%가 일반적

*💬 이렇게 입력하면 됩니다*

30일 성과 보기:
```
/rinda-ai:rinda 지난 30일간의 캠페인 성과를 보여줘
```

주간 리포트:
```
/rinda-ai:rinda 이번 주 이메일 캠페인 성과를 분석해줘
```

분기 비교 리포트:
```
/rinda-ai:rinda 최근 3개월 성과를 업계 평균과 비교해서 알려줘
```

_조회 기간 옵션: `7d` `2w` `30d` `90d` `3m`_
```
